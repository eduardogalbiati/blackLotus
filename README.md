#Silex seed Application
### Features
  - Doctrine DBAL as DataMapper
  - Firewall Implementation
  - CRUD Permissions for User
  - CRUD Structure
   
  
### Install

  - Database
```
    docs/doctrine/banco+data.sql
```
  - Configuration
```
    src/config.php
```
  - Application
```
    PHP -S localhost:9999 -t /web
```
